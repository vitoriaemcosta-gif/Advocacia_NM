# ⚖️ Nelemar Machado - Advocacia Especializada

> **Status do Projeto:** Concluído 🚀

Landing page profissional desenvolvida para o escritório de advocacia **Nelemar Machado**, com atuação especializada nas áreas de Direito Trabalhista, Previdenciário e de Família. 
O projeto apresenta uma interface elegante, responsiva e focada na experiência do usuário e na conversão de novos clientes.

---

## ✨ Demonstração Visual

Abaixo está uma visão geral da interface e do fluxo de contato implementados no projeto:

* **Página Principal (Desktop & Mobile):** Design minimalista utilizando uma paleta de cores sóbria (azul marinho e dourado).
* **Seções Institucionais:** Espaço dedicado para a trajetória da Dra. Nelemar, Missão, Visão, Valores e Áreas de Atuação.
* **Formulário de Contato Inteligente:** Integração visual para captação de mensagens por e-mail e botão de direcionamento rápido para atendimento via WhatsApp.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web sem a necessidade de frameworks robustos, priorizando a performance e o carregamento rápido:

* **HTML5:** Estruturação semântica de todo o conteúdo para melhor acessibilidade e SEO.
* **CSS3:** Customização de layout em Grid e Flexbox, animações fluidas nos cards e variáveis globais para paleta de cores (`:root`).
* **Vanilla JavaScript:** Gerenciamento do menu hambúrguer interativo para dispositivos móveis e efeitos de navegação.
* **Google Fonts:** Utilização das famílias de fontes *Playfair Display* e *Montserrat*.
* **Ícones:** Biblioteca *Phosphor Icons* (carregamento otimizado via `defer`) e *Font Awesome*.

---

## 🖥️ Funcionalidades do Projeto

* 📱 **Design 100% Responsivo:** Adaptado perfeitamente para visualização em Smartphones, Tablets, Laptops e Desktops.
* 📍 **Efeito de Rolagem Suave:** Navegação interna por âncoras (`scroll-behavior: smooth`) que melhora a experiência de leitura.
* 💬 **Botão de WhatsApp Integrado:** Canal direto e rápido de comunicação configurado com a API oficial do WhatsApp.
* 📩 **Formulário de Contato Estruturado:** Pronto para receber dados de potenciais clientes e descrições breves de casos jurídicos.
* ♿ **Acessibilidade:** Uso correto de tags semânticas, atributos `aria-label` e propriedades de navegação via teclado para o menu mobile.
